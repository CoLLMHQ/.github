# CoLLM

**The AI Agent Platform**

CoLLM is a platform for building, deploying, and managing AI agents at scale. We provide the infrastructure for agent identity, memory, control, and cost management.

---

## Projects

| Project | Description |
|---------|-------------|
| [**collm-index**](https://github.com/CoLLMHQ/collm-index) | Context database for AI agents — persistent, organized memory with tiered loading and filesystem-based retrieval |
| [**collm-registry-service**](https://github.com/CoLLMHQ/collm-registry-service) | Identity, auth, and control plane for AI agents — registration, telemetry, C2 commands, and FinOps |
| [**collm-code**](https://github.com/CoLLMHQ/collm-code) | Multi-provider coding agent CLI — OpenAI, Gemini, DeepSeek, Ollama, and 200+ models |
| [**collm-loop**](https://github.com/CoLLMHQ/collm-loop) | Structured AI-assisted development for Claude Code — Plan-Apply-Unify loop with session persistence and quality gates |
| [**collm-devops**](https://github.com/CoLLMHQ/collm-devops) | Code generation and deployment toolkit for the CoLLM platform |

## What We Build

- **Agent Identity** — Secure registration, authentication, and credential management
- **Agent Memory** — Persistent context with intelligent retrieval and tiered loading
- **Agent Control** — Remote command dispatch, heartbeats, and lifecycle management
- **Agent Observability** — Telemetry ingestion, usage tracking, and cost analytics
- **Agent Governance** — Multi-tenant orgs, teams, budget controls, and audit trails
